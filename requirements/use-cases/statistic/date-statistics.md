# Summary

- **Title:** Date Statistics
- **Description:** System provides information about what happends in all subdates of the date
- **Pre-condition:** Date is valid
- **Pos-condition:** Date statistics are shown

# Flow

### Normal Flow

1. User initiates the date statistics process.
2. User provides a date, such as:
    - No date _(subdate = Years)_
    - Year _(subdate = Months of the year)_
    - Month and Year _(subdate = Days of the month or the year)_
2. The system provides informations about what happened in the subdates, such as:
    - Number of users created
    - Number of trips
    - Number of cancelled trips
    - Percentage of trips with delay
    - Number of passengers
    - Number of unique passengers
